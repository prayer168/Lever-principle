# 槓桿原理互動數位教材（Lever Principle）

適用國小高年級～國中自然科「簡單機械」單元的互動網頁教材。

🌐 線上版：https://prayer168.github.io/Lever-principle/

## 學習目標

1. 指認槓桿三要素：支點、施力點、抗力點。
2. 透過槓桿實驗器模擬，歸納平衡條件「施力×施力臂＝抗力×抗力臂」。
3. 判斷生活工具屬於省力或費力槓桿，並說明省力的代價（費時／費距離）。

## 教材結構（七頁籤）

| 頁籤 | 內容 |
|---|---|
| 🎯 任務 | 情境任務（省力工程師）與三個學習目標 |
| 🔍 探索 | 三要素標示圖、力臂定義、平衡公式 |
| 🧪 互動實驗 | 槓桿實驗器模擬（砝碼×格數）＋平衡挑戰（多解題） |
| 🎬 動畫 | 撬石頭五階段動畫：支點位置與省力的關係 |
| 🏠 生活應用 | 八種工具點擊互動，指認三要素與分類 |
| 🏆 闖關 | 十題選擇題，即答即回饋與解說 |
| 📚 資源 | PhET、均一、因材網、科教館等延伸資源 |

## 技術

- 純靜態 HTML / CSS / JavaScript，無框架、無建置步驟。
- 原創 SVG 圖與動畫；支援鍵盤操作、`aria-live` 狀態回報與 `prefers-reduced-motion`。
- 響應式支援桌機（1440×1000）、平板（768×1024）、手機（390×844）。

## 本機使用

直接以瀏覽器開啟 `index.html` 即可，無需安裝任何套件。

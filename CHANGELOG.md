# Changelog

## [1.0.0] - 2026-07-14

### 新增

- 七頁籤教材架構：任務、探索、互動實驗、動畫、生活應用、闖關、資源。
- 探索頁：槓桿三要素標示圖（SVG）、力臂定義與平衡公式。
- 互動實驗：槓桿實驗器模擬（左右各 10 格、1–10 個砝碼、力矩即時計算、砝碼保持鉛直）、五題輪替的平衡挑戰（接受所有等力矩解）。
- 動畫頁：撬石頭五階段動畫（三要素 → 施力 → 移支點 → 省力比較），含播放／暫停／上一步／下一步／重播。
- 生活應用：八種工具互動卡（蹺蹺板、剪刀、拔釘鎚、開瓶器、獨輪手推車、胡桃鉗、鑷子、釣竿），點擊顯示動作與三要素分類。
- 闖關：十題選擇題，即時回饋、解說與成績評語。
- 資源頁：PhET、均一、因材網、科教館連結（含來源、適用年級、查核日期）。
- 無障礙：頁籤鍵盤導覽、`aria-live` 狀態、`prefers-reduced-motion` 支援、SVG `role="img"` 與替代文字。
- 社群分享：1200×630 Open Graph 預覽圖與完整 og/twitter metadata。
- 文件：README、references.md（內容查核）、test-report.md（測試報告）。

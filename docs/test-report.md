1. **頁籤切換**：七個頁籤點擊切換正常；`aria-selected`、`hidden` 同步；支援 ←→/Home/End 鍵盤導覽（程式碼實作，桌機驗證）。✔
2. **主控台**：無任何錯誤或警告（onlyErrors 檢查為空）。✔
3. **水平溢出**：手機 390px 下逐頁檢測，7 頁籤溢出元素數皆為 0。✔
   - 註：第一輪 Edge headless 於 390px 出現的「溢出」為 headless 視窗最小寬度 500px 的裁切假象，經真實瀏覽器 375/390px 實測排除。
4. **探索頁圖**：三要素標籤、引線、力臂尺寸線互不重疊，標籤位於主體外。✔
5. **互動實驗**：
   - 預設 4×3＝4×3 平衡，橫桿水平、狀態列綠色。✔
   - 右側改 9×8＝72 對 12：向右傾斜、狀態列紅字、力矩讀數正確、砝碼堆保持鉛直（反向旋轉補償）。✔
   - 砝碼 10 個時不觸地、數量標籤可見（修正砝碼尺寸後重測）。✔
   - 平衡挑戰：左側鎖定、提示文字更新、達成任一等力矩組合即回報成功（多解皆接受）。✔
6. **動畫**：五階段（0–4）逐步播放；棒尖插於石下、施力端翹空中；下壓時石頭抬起、棒身不穿透地面（重新設計幾何後重測通過）；階段字幕與標註同步；播放/暫停/上一步/下一步/重播有效；切換頁籤時自動停止計時器。✔
7. **生活應用**：八張卡片渲染正常、三色圓點清晰；點擊後顯示動作動畫與分類說明（`aria-expanded` 同步）。✔
8. **闖關**：作答即回饋（正解綠、誤答紅並標示正解）、答後鎖定、得分即時更新、重新挑戰可重置；十題答案經人工驗算（見 references.md）。✔
9. **占位掃描**：`待補|範例題|示範題|占位|example.com|TODO|FIXME` 全案無符合。✔
